# symb
```
tip> symb
usage: symb x y [(opt)]
```
put a symbol mark at the point (x,y).

## option
### option with single argument
- `sc:` symbol color
- `st:` symbol style
- `ss:` symbol size, (default is 1)

## list of symbol name

```
tip> opt symbol
names of the symbol :
 non(0), dot(1), plus(pls,2), star(str,3), arc(4), cross(crs,5), square(sqr,6),

 diamond(dia,11), star2(st2,12), square3(sq3,13), cross2(cr2,14), star3(st3,15),
 blacksqr,(bsq,16)
```
