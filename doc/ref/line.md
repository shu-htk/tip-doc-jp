# line
```
tip> line
usage: line x0 x1 y0 y1 [(opt)]
```
draw a line from (x0,y0) to (x1,y1),
where x0, x1, y0, y1 are scalar value.

## option

- `lc:` set line color
- `lt:` set line style
- `lw:` set line width, 0=not draw, 1=draw
- `ae:` draw arrow edge(0=non,1=end,2=begin,3=both)
