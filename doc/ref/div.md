# div
```
tip> div
usage: div nx ny [(bo:0|1)]
divide graph area by horizontal(nx) and vertical(ny)
option:
 bo:0  : horizontal box ordered (default)
 bo:1  : vertical box ordered
```

divide drawing frame nx in horizontal, ny in vertical.

## option

- `bo:` set box drawing order, (0=horizontal, 1=vertical)


see also [box](box.md)
