# fbox
```
tip> fbox
usage: fbox x0 x1 y0 y1 [(opt)]
 draw filled box.
```

draw fill-area of box shape in 2D graph

- x-range is from x0 to x1
- y-range is from y0 to y1

## option

- `fc:` fill area color
- `ft:` fill area style
- `lc:` line color of border
- `lt:` line style of border
- `lw:` line width, (0=not draw, default is 1)
