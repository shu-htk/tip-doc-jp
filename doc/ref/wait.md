# wait
```
tip> wait
usage: wait [t]
 wait t seconds, (t is floating point number)
 if t=0 wait console input
```

example to wait console input
```
tip> do n 1 10; prn [n]; wait 0; end
1
'b':break, others:continue >
2
'b':break, others:continue >
3
'b':break, others:continue > b  ;# break here
tip> 
```
