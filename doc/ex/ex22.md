# ex22: Gain curve of Low Pass Filter
```
@ pi2 = 3.14159265*2
@ C = 10e-9  ;# capacitance [F]
@ R = 50     ;# resistance  [Ohm]
@ f_cut = 1/([pi2]*[R]*[C])       ;# cut-off frequency [Hz]

set freq = range(10000,1e3,10e6)  ;# range from 1[kHz] to 10[MHz]
set gain = 1/sqrt(1+pow([pi2]*freq*[C]*[R],2))
set gain_dB = 20*log10(gain)      ;# (dB of power)

fmt "%.2e"
title "Gain curve of Low Pass Filter (f_cut=[f_cut]Hz )"
plot freq gain_dB (gd:1 xm:0 xs:log lc:red)

line 1e3 10e6 -3 -3 (lt:dot lc:blue)
line [f_cut] [f_cut] -10 -3 (lc:blue ae:1)
text 1e5 -12 "f_cut at -3dB" (tc:blue)
```
